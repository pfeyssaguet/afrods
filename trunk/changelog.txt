AfroDS
----------

31/01/2010 - version libnds
- migration en libnds : refonte de tout
- module splash screen
- module menu : choix des persos
- module game : déplacement du perso sur une map
  * on peut entrer/sortir du mode sélection avec Start
  * en mode sélection, on peut basculer entre l'inventaire et l'équipement avec R/L
  * en mode sélection, on peut équiper, retirer ou utiliser un item avec A (suivant le contexte)
  * en mode map, on peut lancer un combat en appuyant sur X
  * en mode map, on peut entrer dans une maison en allant sur la porte et en appuyant sur A

- module battle : gestion du combat
  * bouton A pour valider les choix
  * bouton Start pour gagner automatiquement le combat
